# The Rasaganga
The real restaurant ‘The Rasaganga’ stuck in my head so much that I just had to borrow the name for my project!

A modern, responsive restaurant landing page built using HTML, CSS, and JavaScript.

## Features

- Hero section with animated text and images using ScrollReveal.js
- Signature dishes and feature cards displayed in a responsive grid layout
- About and Chef sections with interactive content and animations
- Testimonial slider powered by Swiper.js with autoplay and clickable pagination
- Mobile-friendly navigation with a toggle menu icon
- Smooth scroll animations and hover effects for enhanced UX

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Variables, Responsive Design)
- JavaScript (DOM Manipulation, Event Listeners)
- ScrollReveal.js for scroll animations
- Swiper.js for testimonial slider
- RemixIcon for icons
